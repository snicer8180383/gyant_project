# gyant_project

## Intallation

### Server

```bash
cd server

npm instal

npm run dev
```

### Client

```bash
cd client

npm instal

npm start
```

## Configuration

If you want to change the server port you can go to the index.js file inside server folder and there is a variable to set the port.

#### Example

```javascript
...

const port = 1337;

...
```
